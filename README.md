# 飞机大战1.0

一款基于 HTML5 Canvas 开发的飞机大战游戏。

## 游戏特性

- 🎮 对战AI模式：挑战32关，每4关一个BOSS
- 👥 双人对战模式：本地双人对抗
- 🔫 4种武器系统：普通弹、导弹集束、粒子散射、激光束
- 💣 核弹技能：全屏轰炸（AI模式）
- 📱 移动端支持：虚拟方向键和触摸按钮
- 📈 成长系统：击杀敌人获得经验和金币

## 操作说明

- **WASD**：移动飞机
- **1-4**：切换武器模式
- **K**：释放核弹（AI模式）
